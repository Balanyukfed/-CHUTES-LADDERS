Аккордеон

Вертикально сложенный список элементов, таких как текстовые метки или эскизы. Каждый элемент списка может быть «развёрнут» или «раскрыт», чтобы показать содержание, связанное с этим элементом.

Предоставляет большие объемы контента в небольшом пространстве за счет постепенного раскрытия. Заголовок дает пользователю общий обзор содержимого, позволяя пользователю решить, какие разделы читать. Этот компонент облегчает обработку и поиск информации.

Когда использовать?

  • Для систематизации сопутствующей информации.
  • Для сокращения длины страницы и уменьшения прокрутки, когда контент не имеет решающего значения для полного прочтения.
  • Когда пространство ограничено и длинный контент не может отображаться сразу, например, в мобильном интерфейсе или на боковой панели.

Макет

По умолчанию элементы аккордеона закрыты. Значок с шевроном указывает на взаимодействие с элементом. Шеврон может располагаться как в начале, так и в конце заголовка. Необходимо согласовывать размещение значка между элементами.

Содержание

Значок с шевроном обозначает дополнительный контент. Заголовок в виде аккордеона должен давать представление о содержимом панели. Заголовок должен быть кратким.
