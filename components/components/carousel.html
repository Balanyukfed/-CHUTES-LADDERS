Карусель

Карусель — это тип шаблона пользовательского интерфейса, который отображает несколько элементов контента в линейной последовательности. Карусель может содержать различные виды контента, включая текст, изображения и мультимедиа.

Карусель создается для размещения большего количества элементов. Элементы в последовательности, которые выходят за пределы контейнера карусели, скрыты от просмотра.

Анатомия

Карусель в большинстве случаев включает визуальную навигацию (например, кнопки и / или стрелки), указывающую, что компонент является интерактивным и может управляться непосредственно пользователем.

Пользователь контролирует, какие элементы карусели будут ему видны, для навигации слева направо или наоборот присутствуют кнопки “Назад” / “Вперед”.

При наведении курсора на карточку ее размер изменяется.

Параметры

Компонент карусель использует "карточки" для отображения вложенного содержимого в линейной последовательности. Непрерывная группа видимых элементов составляет страницу.

Руководство

Карусель можно настроить на автоматическую перелистывание. Также помимо смены элементов с помощью клика на навигационные кнопки “вперед”/ “назад”, элементы можно перемещать с помощью свайпа.
