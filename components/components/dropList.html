Выпадающие списки

Выпадающие списки представляют собой список опций, из которых пользователь может выбрать один или несколько вариантов.

Анатомия

Раскрывающиеся списки включают в себя название, информирующее пользователя о том, чего ожидать в списке, вспомогательный текст, поле и меню опций отображаемых в раскрытом состоянии списка. Название и вспомогательный текст могут помочь пользователю принять обоснованное решение при выборе.

Виды

Существует три различных вида раскрывающихся списков, которые поддерживают различные функции: раскрывающийся список, множественный выбор и поле с возможностью ввода текста.

Раскрывающийся список

Позволяет пользователю выбрать один вариант из списка.

Множественный выбор

Позволяет пользователю выбирать и фильтровать несколько вариантов из списка.

Список с полем ввода

Позволяет пользователю сделать выбор из списка предлагаемых, вероятных или желаемых значений. Меню открывается щелчком мыши в любом месте поля, пользователь может начать вводить текст, чтобы просмотреть список опций. Будут выделены варианты, которые наилучшим образом соответствуют введенным символам.

Руководство

Контейнеры с выпадающими списками должны выравниваться по вертикали с другими компонентами на странице. Пункты списка должны быть отсортированы в логическом порядке.
