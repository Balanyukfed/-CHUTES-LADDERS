Навигация

Продуманная система навигации способствует улучшению пользовательского опыта, делает процесс поиска нужной информации более интуитивным и удобным.

Элементы навигации включают в себя различные компоненты, такие как хлебные крошки, табы, кнопки для перехода на другие страницы и т. д.. Они обеспечивают пользователю комфортное передвижение по веб-сайту, позволяют быстро ориентироваться и находить нужную информацию.

Breadcrumbs

Дополнительная навигация, которая помогает пользователям понять, где они находятся в структуре сайта, и перемещаться между уровнями. Список ссылок в виде цепочки в верхней части страницы, представляющих текущую страницу и ее родительские ссылки, пошагово возвращающие пользователя на домашнюю страницу.

BackButton

Кнопка навигации на предыдущую страницу. Помогает пользователю ориентироваться. Для удобства навигации рекомендуем заменять стандартный текст “назад” на указание раздела, к которому вернется пользователь.

SegmentedControl

Быстрая навигация между сгруппированным контентом без перезагрузки страницы. Переключение состояний или выбор одного варианта из нескольких.

Tabs

Элемент быстрой навигации между сгруппированным контентом без перезагрузки страницы. Отображает в общем контейнере выбранный раздел контента, скрывая остальные разделы.

Группа табов отделяется от контента границей блоков или линией. Один (и только один) таб из группы всегда активен. Клик по неактивному табу скрывает текущий контент и отображает контент выбранного. Яркая полоса переезжает к выбранному табу.

Табы акцентные

Используются для навигации или переключения состояний, выбор одного варианта из нескольких. Они привлекают внимание пользователя, выглядят как кнопки основного действия. Рекомендуется использовать редко, в ограниченном пространстве и не сочетать с классическими табами.

Продумайте навигацию так, чтобы пользователю не пришлось постоянно перемещаться между табами для выполнения одной задачи. Нужная информация для завершения определенного этапа работы должна быть на каждой вкладке.

Табы с прокруткой (основные)

Активный выбранный таб подчеркивается на его ширину. Ширина таба равна ширине надписи.
