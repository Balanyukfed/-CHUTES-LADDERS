Радиокнопка

Radio button (радиокнопка) - это элемент интерфейса, который позволяет пользователю выбрать один вариант из нескольких предложенных. Обычно радиокнопки используются в формах для выбора определенной категории, настроек или параметров.

Анатомия

Радиокнопки всегда образуют группы, их следует использовать вместо чекбоксов, если из списка нужно выбрать только один элемент.

Руководство

Радиогруппы могут быть выровнены по вертикали или горизонтали. Но их допустимо располагать в одну строку, только если два варианта. При количестве вариантов ответа больше двух, радиокнопки располагаются вертикально. Заголовок может располагаться как сверху, так и слева от кнопок.
