Степпер

Степпер используется для выбора количества экземпляров каждой позиции магазина, которые пользователь захочет приобрести. Например, пользователь может использовать степпер для выбора количества игр, которые он хочет добавить в корзину перед оформлением покупки. Таким образом, степпер позволяет удобно и точно указывать желаемое количество товаров.

Анатомия

Степпер состоит из двух кнопок. При клике на одну из кнопок увеличивает или уменьшает контролируемое значение . На кнопках иконки “+” и “-”. Степпер может быть как стандартного, так и акцентного цвета. Кнопка акцентного цвета нужна для привлечения внимания, на экране должно быть не больше одного акцентного степпера.

Поведение

Степпер используется в узком диапазоне для задач, которые можно решить в несколько кликов, для широких диапазонов лучше использовать слайдеры. У степперов есть допустимый диапазон значений.

Кнопка блокируется, если будет достигнуто минимальное или максимальное допустимое значение диапазона.
