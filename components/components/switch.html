Переключатель

Переключатель (switch) - это элемент пользовательского интерфейса, который позволяет пользователям выбирать между двумя вариантами: включено/выключено, активно/неактивно. Переключатели обычно представляют собой горизонтальные ползунки с двумя состояниями, которые можно менять при помощи клика.

Анатомия

Переключатели мгновенно выполняют действие. Их нельзя использовать для выбора пунктов в списке из нескольких вариантов.

В большинстве случаев переключатели используются с надписями. Надписи могут располагаться слева или справа от переключателя. Если вы используете переключатель без надписи, он должен отображаться в понятном контексте.

Руководство

Если подписи расположены слева, нужно выровнить все переключатели по правой границе. Если список состоит из нескольких переключателей, подпись нужно располагать с одной стороны.
